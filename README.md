# Hermes Scripts

Custom scripts and utilities for Hermes Agent.

## Structure

```
scripts/
  context-status.py       # Context status footer script
  extract_trafilatura.py  # Web extraction with trafilatura
  extract.ps1             # Fallback regex extraction
  hermes-backup.sh        # Backup script
  hermes-restore.sh       # Restore script
  ...
README.md
```

## Usage

Scripts are sourced from `~/.hermes/scripts/`.
